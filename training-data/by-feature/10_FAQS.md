## Troubleshooting Guide

### Contact Form Issues

**Problem: Form submission fails**
1. Check email address contains @ symbol
2. Ensure all fields are filled
3. Check internet connection
4. Try refreshing page and resubmitting
5. If persistent, email directly: contact@aasimshah.com

**Problem: Not receiving confirmation email**
1. Check spam/junk folder
2. Wait up to 24 hours for delivery
3. Verify email address was typed correctly
4. Add contact@aasimshah.com to contacts
5. Try submitting again with different email

### Calendar Scheduling Issues

**Problem: Calendar doesn't load**
1. Allow pop-ups for the website
2. Check JavaScript is enabled
3. Clear browser cache and cookies
4. Try different browser
5. Disable browser extensions temporarily
6. Alternative: send email to schedule

**Problem: No available time slots**
1. Try different dates or weeks
2. Verify time zone in calendar settings
3. Email to request custom time
4. Check "Available For Work" status on homepage

### Website Display Issues

**Problem: Page layout looks broken**
1. Refresh page completely
2. Update browser to latest version
3. Disable browser extensions
4. Try incognito/private mode
5. Clear cache and cookies

**Problem: Images not loading**
1. Wait for page to fully load
2. Refresh page (Ctrl+R or Cmd+R)
3. Clear browser cache
4. Check internet connection speed

### Payment Issues

**Problem: Payment button doesn't work**
1. Enable third-party cookies in browser
2. Try different payment method
3. Contact SafePay support if using SafePay
4. Email to arrange alternative payment
5. Try different browser or device

**Problem: Payment made but no confirmation**
1. Check bank/card statement for transaction
2. Wait 24 hours for confirmation email
3. Check spam/junk folder
4. Email contact@aasimshah.com with transaction date, amount, payment method, and reference number
5. Response within 24–48 hours

### Video Call Issues

**Problem: Cannot join scheduled meeting**
1. Check spam folder for calendar invitation
2. Verify meeting time in your time zone
3. Check calendar invitation for correct link
4. Email 15 minutes before meeting if link not working
5. Alternative: Call phone number provided

**Problem: Technical issues during call**
1. Turn off video, use audio only
2. Close other programs/tabs
3. Switch to phone call if persistent
4. Reschedule if necessary

## General Troubleshooting Tips

**Before Contacting Support:**
- Use updated browser (Chrome, Firefox, Safari, Edge)
- Ensure stable internet connection
- Disable ad-blockers for this website
- Allow cookies for proper functionality
- Check spam folder for emails

**When Reporting Issues:**
- Describe what you see
- Share error messages (screenshots helpful)
- Note when issue started
- Mention device and browser used
- List steps already tried
